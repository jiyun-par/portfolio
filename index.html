  웹접근성 준수 POINT

1. 반복텍스트 숨김 aria-hidden
<Tooltip의 제목 텍스트>

Meaning :
Screen Reader에서 tooltip에 들어가는 텍스트와, 본문 제목태그로 들어가는 텍스트의 중복 읽기를 방지하기 위해 aria-hidden 속성을 추가

Processing Method:
script에서 tooltip 속성에 attr('aria-hidden' ,'true')를 추가하여 제어

2. Modal 접근성
<Video가 들어간 Modal>

Meaning :
스크린리더에서 모달 이외의 모든 콘텐츠는 읽어주지 못하도록 처리

Processing Method:
현재 보여지는 모달 상세내용을 제외한 나머지 모두를
aria-hidden과 inert 속성으로 차단하고,
모달 열기 닫기 버튼 클릭시 포커스 강제 이동 제어
